# Password Manager

A secure command-line password manager built with Python that encrypts and securely stores your passwords using modern cryptography.

## Features

- Encrypt passwords (never store plaintext)
- Master password authentication
- CRUD operations (Create, Read, Update, Delete)
- Search passwords
- Export/Import encrypted vault
- Clean project structure
- Unit tests
- Documentation
- Requirements file
- GitHub-ready

## Commands

1. Add Password
2. View Passwords
3. Search Passwords
4. Update Password
5. Delete Password
6. Copy Password
7. Export Vault
8. Import Vault
9. Change Master Password
10. Exit

## Technologies

- Python
- SQLite
- cryptography (Fernet)
- bcrypt
- argparse
- pytest
